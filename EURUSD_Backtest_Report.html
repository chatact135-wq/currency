Backtest Assumptions

  • BUY entry uses ASK close. BUY exit uses BID high/low.
  • SELL entry uses BID close. SELL exit uses ASK high/low.
  • Spread is included because we used BID and ASK separately.
  • If stop and target occur in the same M1 candle, this test counts stop first, which is conservative.
  • This test uses M1 candles, not tick-by-tick data, so exact intra-minute order can still be imperfect.

Professional Conclusion

The current idea of fast breakout/reversal detection needs more work. The data shows many whipsaws and late entries. The next system should not be deployed as a live signal engine yet. We need:

  1. More data, especially GBP/USD and more EUR/USD days.
  2. Tick data if possible for fast scalping.
  3. Session filters and news-time filters.
  4. A better strategy state machine that separates momentum continuation from exhaustion/reversal.
  5. Walk-forward testing: build on one period and validate on another period.
